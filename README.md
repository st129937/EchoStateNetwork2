# Сети с эхо-состоянием: анализ памяти и геометрии резервуара

## Описание
Проект реализует сеть с эхо-состоянием на Python. Исследуются:
- свойство эхо-состояния;
- затухающая память и ёмкость памяти;
- влияние геометрии резервуара (случайный, ортогональный, циклический);
- предсказание временных рядов NARMA-10 и Mackey–Glass;
- абляционный анализ гиперпараметров.

## Структура
- `esn_core.py` – класс `EchoStateNetwork` (инициализация, обновление, обучение считывателя).
- `data_utils.py` – генерация рядов, метрики, baseline AR(15).
- `main.py` – запуск всех экспериментов.
- `requirements.txt` – зависимости.
- `weights/` – папка для сохранения обученных моделей.

## Установка и запуск
1. Клонировать репозиторий.
2. Установить зависимости.
3. Запустить все эксперименты: "python main.py"
Результаты сохраняются в CSV-файлы и PDF-графики.
